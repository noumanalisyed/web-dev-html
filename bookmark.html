HTML Links - Create Bookmarks HTML links can be used to create bookmarks, so that readers can jump to specific parts of a web page. Create a Bookmark in HTML Bookmarks can be useful if a web page is very long. To create a bookmark - first create the bookmark, then add a link to it.
